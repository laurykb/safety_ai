# Audit des fichiers

## Structure principale

- `run.py` : Point d'entree
- `streamlit_app.py` : Application Streamlit
- `app/ui/` : Onglets (tab_operations, tab_inference, etc.)
- `src/cyberbullying/` : Logique metier (loading, embedder, models)
- `scripts/` : Scripts CLI (run_pipeline, download_research_datasets, check_duplicates)
- `configs/` : Configuration YAML
